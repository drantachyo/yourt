# 🎯 Краткая шпаргалка для добавления 3D модели

## Как использовать промпты для добавления новой модели?

### 1️⃣ **Быстрый способ** (скопировать и заполнить)

```
Добавь на сайт новую интерактивную 3D модель:

**Название:** Боевой Конь Казахский
**GLB файл:** war_horse.glb
**Компоненты модели:**
  - Седло казахское
  - Сбруя и украшения  
  - Броня и оружие

**Описания на русском:**
  - Седло казахское: "Удобное кожаное седло с орнаментом, идеально подходит для дальних путешествий по степи"
  - Сбруя и украшения: "Традиционные украшения из металла и кожи, отражающие статус воина"
  - Броня и оружие: "Защитное вооружение казахского воина, инкрустированная золотом"

**Аудио:** нужны файлы для всех трёх компонентов (RU, KZ, EN)
**Позиция:** новая секция перед галереей
```

### 2️⃣ **Требуемые файлы**

✅ **Основной файл:** `model_name.glb` (50-200K полигонов, <5MB)

✅ **Аудио описания** (опционально):
```
component1_ru.mp3   component1_kz.mp3   component1_en.mp3
component2_ru.mp3   component2_kz.mp3   component2_en.mp3
component3_ru.mp3   component3_kz.mp3   component3_en.mp3
```

### 3️⃣ **Структура GLB модели**

- Каждый компонент = отдельный объект/группа в GLB
- Разные части должны быть именованы (для raycast выбора)
- Все текстуры встраивать внутрь GLB

### 4️⃣ **Что происходит автоматически**

Когда ты скажешь промпт Copilot:

✅ Я загружу GLB в Three.js  
✅ Автоматически масштабирую и центрирую  
✅ Добавлю меню выбора компонентов  
✅ Настрою освещение и камеру  
✅ Добавлю i18n для всех языков  
✅ Свяжу аудио файлы  
✅ Создам красивую HTML секцию

## 📝 Примеры готовых промптов

### Пример 1: Кибитка
```
Добавь интерактивную 3D модель кибитки (традиционного кочевого дома).

Файл: kibithka.glb
Компоненты: каркас, текстиль, вход_дверь

Описания:
- каркас: "Основной деревянный каркас кибитки, более компактный чем юрта"
- текстиль: "Войлочное покрытие белого цвета"
- вход_дверь: "Резная дверь с узорами"

Позиция: новая секция после юрты
```

### Пример 2: Казахский щит и оружие
```
Добавь на страницу музей казахского оружия - интерактивный 3D модуль.

Файл: kazakh_weapons.glb
Компоненты: щит, меч, лук, колчан

Для каждого - описание, аудио на RU/KZ/EN
Тема: золотая и коричневая цветовая схема
Позиция: в галерее, отдельный блок
```

### Пример 3: Ремесла (станок ткацкий)
```
Добавь 3D модель традиционного ткацкого станка.

Файл: weaving_loom.glb
Компоненты: каркас, нити, челнок, готовая ткань

Описания каждого компонента на русском
Нужны аудио файлы
```

## ⚡ Самые частые ошибки

❌ GLB файл > 10MB → **Оптимизируй текстуры и полигоны**

❌ Компоненты не выбираются → **Проверь, что разные части это отдельные объекты**

❌ Аудио не воспроизводится → **Назви файлы правильно: component_ru.mp3**

❌ Модель перевёрнута → **Поправь ориентацию в 3D редакторе перед экспортом**

❌ Низкий FPS → **Уменьши полигоны, убери лишние текстуры**

## 🚀 Готовые команды для Copilot

### Команда 1: Добавить модель
```
Используя файлы [model.glb] и аудио описания, добавь интерактивную 3D модель на сайт.
Компоненты: [list]
Описания: [descriptions]
```

### Команда 2: Модифицировать существующую
```
Обнови освещение/цвета/масштаб 3D модели [name].
Сделай её более/менее яркой, переделай меню компонентов.
```

### Команда 3: Добавить аудио
```
Добавь озвучку для компонентов 3D модели [name].
Файлы: [list of mp3]
Языки: RU, KZ, EN
```
