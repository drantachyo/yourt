# 🚀 Деплой на Vercel

## Быстрый старт (5 минут)

### Шаг 1: Переименовать главный файл
```bash
# В папке проекта
mv "index (3).html" index.html
```

### Шаг 2: Инициализировать Git репозиторий
```bash
cd yurt_with_voice

# Инициализировать
git init

# Добавить все файлы
git add .

# Первый коммит
git commit -m "Initial commit: Kazakh Yurt Interactive 3D Experience"
```

### Шаг 3: Создать репозиторий на GitHub
1. Перейти на https://github.com/new
2. Назвать репозиторий: `yurt_with_voice`
3. Нажать "Create repository"
4. Скопировать команду для `…or push an existing repository from the command line`
5. Выполнить в терминале

Пример:
```bash
git remote add origin https://github.com/YOUR_USERNAME/yurt_with_voice.git
git branch -M main
git push -u origin main
```

### Шаг 4: Деплоить на Vercel
1. Перейти на https://vercel.com/sign-up (создать аккаунт)
2. Нажать "New Project"
3. Нажать "Import Git Repository"
4. Авторизировать GitHub
5. Выбрать репозиторий `yurt_with_voice`
6. Нажать "Import"
7. Нажать "Deploy"

**Всё готово!** 🎉 Ваш сайт будет доступен на `https://yurt-with-voice.vercel.app` (или похожем URL)

## ❓ Альтернативный способ: Vercel CLI

Если хотите быстрее:

```bash
# 1. Установить Vercel CLI
npm install -g vercel

# 2. Перейти в папку проекта
cd yurt_with_voice

# 3. Деплоить
vercel

# Следовать инструкциям в терминале
```

## 📊 Что включено в конфиг

**vercel.json** оптимизирует:
- ✅ **Кеширование GLB моделей** - 1 год (очень редко меняются)
- ✅ **Кеширование MP3** - 1 год
- ✅ **HTML кеш** - 1 час (свежий контент, но с кешем)
- ✅ **Редиректы** - для правильной работы SPA навигации

---

## 🔧 Проверки перед деплоем

Убедиться что:
- ✅ `index.html` находится в корне папки
- ✅ Все GLB файлы на месте (`*.glb`)
- ✅ Все MP3 файлы на месте (`*.mp3`)
- ✅ Git репозиторий создан и залит на GitHub
- ✅ Vercel аккаунт создан

---

## 🌐 Домен (опционально)

После деплоя можно:

1. **Добавить собственный домен**
   - В Vercel проекте → Settings → Domains
   - Добавить домен и следовать инструкциям
   
2. **Купить домен**
   - Vercel интегрирован с доменными регистраторами
   - Или купить у любого провайдера (Namecheap, Google Domains и т.д.)

## 🆘 Частые проблемы

| Проблема | Решение |
|----------|---------|
| "Cannot find index.html" | Переименовать `index (3).html` в `index.html` |
| "404 на статические файлы" | Убедиться что GLB и MP3 в корне репозитория |
| "3D модель не загружается" | Проверить консоль браузера (F12) на ошибки CORS |
| "Audio не воспроизводится" | Может быть браузер блокирует, требуется взаимодействие пользователя |
